# 전산 포트폴리오 요약

## 저자의 문제 인식 및 주장 (15문장 이내)

저자는 현대 사회에서 지식 공유와 학습 환경의 중요성이 커지고 있지만, 여전히 시간적·공간적 제약으로 인해 학습 기회에 접근하기 어려운 사용자들이 존재한다고 인식하고 있다. 이를 해결하기 위해 누구나 선생님이 될 수 있고, 누구나 학생이 될 수 있는 지식 공유 플랫폼을 목표로 HomeSchools 프로젝트를 기획하였다. 이 프로젝트는 실시간 채팅과 1:1 화상 수업 기능을 제공하는 웹 기반 학습 지원 서비스로, 사용자 역할에 제한이 없는 학습 참여 구조를 제공하고자 한다. 또한, 실시간 화상·채팅을 통한 몰입도 높은 소통 환경을 구현하고, 사용자 의견을 서비스 개선에 반영할 수 있는 구조를 설계하였다. 기술적으로는 React와 Spring Boot를 활용하여 Front-End와 Back-End를 분리하고, WebRTC와 WebSocket을 통해 실시간성을 확보하였다. 또 다른 프로젝트인 Dog-Go는 반려견 보호자와 산책 서비스를 연결하는 웹 기반 예약 시스템으로, 사용자와 관리자의 역할을 명확히 분리하고, Spring Security 기반의 인증 방식을 적용하여 보안성을 강화하였다. 이 프로젝트는 관리자 중심의 운영 관점 기능을 강조하며, 실 배포 환경에서의 인증 및 보안 구조를 적용하였다. 저자는 이러한 프로젝트를 통해 인증·보안 구조에 대한 이해, 실시간 통신 및 네트워크 기반 서비스 설계 경험, 운영 환경에서의 문제 분석 및 해결 능력을 얻었다고 주장한다.

## 저자 소개

신승철은 웹 표준 코딩 및 웹 프로그래밍, 데이터베이스 설계, 시스템 운영 경험을 갖춘 개발자이다. 그는 React 기반의 SPA 개발, REST API 연동, 비동기 통신 처리, CORS/HTTPS 환경에서의 서비스 운영 경험을 보유하고 있다. 또한, PostgreSQL 기반의 테이블 설계 및 관계 모델링, Spring Data JPA ORM을 활용한 데이터 처리, 사용자 및 관리자 권한 분리 설계, HttpSession 및 JWT 인증 정책 설계 경험을 가지고 있다. 신승철은 HomeSchools와 Dog-Go 프로젝트를 통해 실시간 통신 및 네트워크 기반 서비스 설계, 인증 및 보안 구조 이해, 운영 환경에서의 문제 해결 능력을 발전시켰다.
